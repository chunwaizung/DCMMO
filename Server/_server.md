## web
- install python3
- 创建django应用
    - django-admin startproject DCWebApp
- 创建子app
    - python manage.py startapp polls
- 配置django
    - config django
- 初始化数据库
    - python manage.py migrate
- 更新自定义模块
    - python manage.py makemigrations polls
- 迁移数据库
    - python manage.py sqlmigrate polls xxxx
- 创建表
    -  python manage.py migrate
